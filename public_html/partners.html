Crypti
We are happy to announce a parnership between the Sia project and the Crypti project. For those who don't know, Crypti is a decentralized application platform that empowers developers to write their own applications. Crypti provides a full stack solution for deploying truly decentralized applications onto the blockchain, while also providing an excellent user expereince for anyone looking to make safe and easy transactions within seconds. Crypti's very own Dapp store gives consumers the same if not superior interface consumers have come to expect from existing centralized solutions currently dominating the market. Each Dapp operates on a separate custom sidechain secured by Crypti's own version of the Delegated Proof of Stake consensus model. For the progressive startup looking for the perfect platform on which to launch their first Dapp, Crypti is the number one choice. Further development of the core protocol is maintained by the Crypti Foundation, a dedicated team of developers, entrepreneurs and engineers from across the globe.
